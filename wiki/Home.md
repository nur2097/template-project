## 📈 Project Stats

- **Language**: TypeScript
- **Framework**: NestJS 10.x
- **Database**: PostgreSQL (Primary), MongoDB (Logs)
- **Cache**: Redis
- **Authentication**: JWT + Refresh Tokens
- **Authorization**: RBAC with Custom Permissions
- **Monitoring**: Winston + OpenTelemetry
- **Testing**: Jest
- **Documentation**: Swagger/OpenAPI
